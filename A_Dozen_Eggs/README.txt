How to run this application:
1. Install Oracle DB, Python, and Flask
2. In the console, put yourself in the inner flaskr folder (ICS321Final\src\flaskr\flaskr)
3. Run the schema.sql in your database
4. In flaskr.py, change the username and password to match your own database account information
5. Type "set FLASK_APP=flaskr.py" without the quotations
6. (OPTIONAL) Type "set FLASK_DEBUG=true" without the quotations
7. Type "flask run" without the quotations
8. In your browser, type "127.0.0.1:500" without the quotations.
